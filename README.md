# EHR Database
This project began as a relational database assignment for a database systems course and is being expanded into a portfolio project demonstrating the design and development of a simplified Electronic Health Record (EHR) system.

The goal is to evolve this project from a foundational database into a more realistic healthcare application aligned with interoperability standards and supported by a functional user interface.

This repository documents the ongoing development process, design decisions, and implementation of features over time.
